# CODESOFT

This repository contains the projects I completed during my Python Programming Internship at CodSoft. The internship helped me improve my programming skills and understand how Python can be used to build practical applications.

During this internship, I worked on different beginner-friendly projects that focus on problem solving, logic building, and creating simple user interfaces. Some projects are command-line based while others include GUI interfaces using Python libraries.

Projects Included
1. Password Generator

A Python application that generates strong and random passwords based on the user’s required length. It combines letters, numbers, and special characters to create secure passwords.

2. To-Do List Application

A simple application that helps users manage their daily tasks. Users can add tasks, view tasks, and remove completed tasks.

3. Simple Calculator

A calculator built using Python that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It includes a simple and user-friendly interface.
